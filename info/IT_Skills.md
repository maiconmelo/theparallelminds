Here are the 10 top categories of **IT technical skills** (excluding HPC-specific skills like MPI, Slurm, and parallel filesystems), aggregated from the job descriptions with clear, descriptive category names:

---

### 1. Linux & Unix Systems Administration
The foundational skills for managing, configuring, and troubleshooting Linux-based operating systems in enterprise and research environments.

- Linux administration (RHEL, CentOS, Rocky, Ubuntu, SLES, Debian)
- OS installation, configuration, and patching
- Kernel tuning, systemd, and boot procedures
- Package management (RPM, APT)
- User and group administration
- System performance monitoring and tuning
- Shell environments and command-line proficiency

---

### 2. Networking & Infrastructure
The knowledge of traditional and modern networking technologies that underpin all distributed systems and data center operations.

- TCP/IP fundamentals (routing, switching, VLANs)
- DNS, DHCP, LDAP, NFS, HTTP/HTTPS
- Firewalls, load balancers, VPNs, and proxy servers
- Network security (Zero Trust, network segmentation)
- Software-Defined Networking (SDN) concepts
- Network troubleshooting tools (Wireshark, tcpdump, iptables)
- Structured cabling and data center networking design

---

### 3. Security & Compliance
The practices and technologies required to secure infrastructure, enforce access control, and meet regulatory standards.

- Identity and Access Management (IAM, LDAP, Active Directory, Kerberos)
- Multi-Factor Authentication (MFA) and SSO
- Role-Based Access Control (RBAC) and least-privilege access
- System hardening (STIGs, CIS benchmarks, DISA)
- Security compliance frameworks (NIST, FedRAMP, HIPAA, DoD 8140)
- Vulnerability management and patching
- Security monitoring and auditing
- Encryption and certificate management

---

### 4. Cloud Computing & Hybrid Infrastructure
Expertise in designing, deploying, and managing workloads across public cloud providers and hybrid on-premises/cloud environments.

- Public cloud platforms: AWS, Azure, GCP, OCI
- Cloud provisioning and resource management
- Virtual networking in the cloud (VPCs, subnets, security groups)
- Cloud storage (object storage like S3, block storage, cloud file systems)
- Hybrid cloud architectures and integration
- Cloud cost optimization and capacity planning
- Cloud IAM and identity federation

---

### 5. Containerization & Orchestration
The skills required to package, deploy, and manage applications using containers and container orchestration platforms.

- Containerization: Docker, Podman, containerd
- Container orchestration: Kubernetes (vanilla, OpenShift, Rancher, EKS/AKS/GKE)
- Helm charts and Kustomize for Kubernetes packaging
- Container networking (CNI plugins like Calico, Cilium, Multus)
- GitOps workflows (ArgoCD, FluxCD)
- Operator frameworks and custom controllers
- Container security and image scanning

---

### 6. Automation & Infrastructure as Code
The ability to automate infrastructure provisioning, configuration, and lifecycle management using modern tooling.

- Configuration management: Ansible, Puppet, Chef, SaltStack
- Infrastructure as Code: Terraform, OpenTofu, CloudFormation
- Bare-metal provisioning: xCAT, Foreman, Metal3, Ironic, MAAS
- Image management: Packer, Kickstart
- Secrets management: HashiCorp Vault
- Automation scripting (Python, Bash) for operational tasks

---

### 7. CI/CD & Software Delivery
The practices and tools used to build, test, and deploy infrastructure and applications in a repeatable, reliable manner.

- CI/CD pipelines: GitLab CI, Jenkins, GitHub Actions
- Version control: Git (GitHub, GitLab, Bitbucket)
- Test automation and validation
- Continuous Integration/Continuous Delivery methodologies
- Infrastructure testing and validation
- Deployment strategies (rolling, blue-green, canary)
- Artifact management and registries

---

### 8. Monitoring, Observability & Telemetry
The ability to collect, visualize, and analyze data to ensure system health, performance, and reliability.

- Metrics collection: Prometheus, Datadog, Nagios, Zabbix
- Log management: ELK Stack (Elasticsearch, Logstash, Kibana), Splunk, Loki
- Visualization: Grafana, Kibana
- Distributed tracing: OpenTelemetry, Jaeger
- Alerting and incident management
- Service Level Objectives (SLOs) and Service Level Indicators (SLIs)
- Capacity and performance reporting

---

### 9. Scripting & Programming
The practical coding skills needed to automate tasks, build tooling, and integrate systems.

- Scripting: Bash, Python, Perl, Ruby, PowerShell
- Programming languages: Go, C/C++, Java, Rust
- API development and integration (REST, gRPC)
- SQL and database querying (PostgreSQL, MySQL)
- Data processing libraries (pandas, NumPy) for operational analytics
- Code quality practices (testing, code reviews, version control)

---

### 10. Data & Storage Infrastructure
The knowledge of enterprise storage technologies, data lifecycle management, and backup/recovery strategies. *(Parallel filesystems like Lustre and GPFS are excluded from this list)*

- Storage protocols: NFS, SMB/CIFS, iSCSI, S3
- Storage architectures: SAN, NAS, object storage, DAS
- Storage arrays and appliances: NetApp, Dell EMC, Pure Storage, DDN
- Data backup and disaster recovery strategies
- Data lifecycle management and archival
- Storage capacity planning and performance tuning
- Storage monitoring and health management
